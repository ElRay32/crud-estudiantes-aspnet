# crud-estudiantes-aspnet
# CRUD de Estudiantes - ASP.NET Core MVC

Este proyecto es un CRUD funcional de estudiantes desarrollado en C# con ASP.NET Core MVC.

## Funcionalidades
- Crear estudiantes
- Listar estudiantes
- Editar estudiantes
- Eliminar estudiantes

## Tecnologías utilizadas
- C#
- ASP.NET Core MVC
- Entity Framework Core
- SQLite
- GitHub Desktop
- Git Flow

## Ramas utilizadas
- main
- developer
- qa
- feature/project-setup
- feature/student-model
- feature/create-student
- feature/list-edit-student
- feature/delete-student
